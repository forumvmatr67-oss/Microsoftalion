# Microsoftalion: новое число для науки и образования

**Microsoftalion** — это математический термин для числа $10^{10\,000}$ (единица с 10 000 нулями). Проект предлагает официально использовать этот термин в образовательных инициативах Microsoft и за их пределами.

## Что это за число?

* **Запись**: $10^{10\,000}$.
* **Визуально**: единица и за ней 10 000 нулей.
* **Сравнение**: превосходит гугол ($10^{100}$) в $10^{9\,900}$ раз.
* **Применение**: теория чисел, криптография, квантовые вычисления, визуализация сверхбольших величин.

## Зачем нужен этот термин?

* Популяризация математики среди молодёжи.
* Инструмент для изучения экспоненциального роста.
* Возможность визуализировать масштаб в симуляциях (например, в Minecraft Education Edition).
* Связь бренда Microsoft с инновациями в науке.
