# Образовательные задачи с Microsoftalion

**Для школьников (12–15 лет)**:
* Сколько времени потребуется, чтобы записать Microsoftalion вручную (если писать 1 цифру в секунду)?
* Сравните размер Microsoftalion с количеством атомов во Вселенной (~$10^{80}$).


**Для студентов (16+ лет)**:
* Оцените, сколько памяти потребуется для хранения Microsoftalion в двоичном виде.
* Напишите алгоритм, который выводит первые и последние 100 цифр Microsoftalion.
* Создайте симуляцию «мир Microsoftalion» в Minecraft (используя API).
