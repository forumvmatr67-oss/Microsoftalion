
### 2. CONTRIBUTING.md

Краткие правила для потенциальных участников:

```markdown
## Как внести вклад в проект Microsoftalion?

Вы можете помочь:
* добавить примеры кода на других языках;
* создать визуализацию числа (графики, анимации);
* написать образовательные задачи с использованием Microsoftalion;
* перевести документацию на другие языки.

**Порядок действий**:
1. Сделайте fork репозитория.
2. Создайте ветку для вашей идеи (`git checkout -b feature/AmazingFeature`).
3. Зафиксируйте изменения (`git commit -m 'Add some AmazingFeature'`).
4. Отправьте ветку в удалённый репозиторий (`git push origin feature/AmazingFeature`).
5. Откройте Pull Request.

Мы рассмотрим ваш вклад и свяжемся с вами!
